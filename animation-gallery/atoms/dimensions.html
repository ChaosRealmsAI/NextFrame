04 · WHY

为啥动 · 意图原子

the intent atoms — what purpose does this animation serve

每个动画都在"说一句话"。6 种主要意图：

✓
反馈 Feedback
告诉用户"这件事成了/失败了/进行中"。确认类是最高频的意图，占 UI 动画的 60%。
✓ success · ✗ error · ⟳ progress
↗
引导 Guide
告诉用户"接下来看这里/做这个"。用注意力 hack 把用户带到该去的地方。
脉冲高亮 · 箭头指向 · 光圈聚焦
♥
情感 Emotion
让用户"感受到"某种情绪。庆祝 / 温暖 / 紧张 / 宁静 / 威严。非理性但极高效。
庆祝粒子 · 慢呼吸 · 震屏
✦
品牌 Brand
让用户记住"这是 XX 的动画"。签名式动效 — 看到就知道是哪家。
Arc Q 弹 · Linear 瞬时 · TikTok 爆发
📊
信息 Information
让"数据变化"可见。数字跳动、图表形变、进度推进 — 把抽象变具象。
CountUp · Data Morph · Progress Ring
📖
叙事 Story
"先告诉你 A · 然后 B · 最后 C"。有起承转合。短视频 / PPT 的灵魂。
Hero → Peak → Resolve
05 · WHEN

何时动 · 时机原子

the timing atoms — the phase of lifecycle

同样一个"动作"，用在不同时机 = 不同动画。6 种主要时机：

🚀
Hero · 开场
首次登场。发布会标题 / 页面首屏 / 视频开头。剧场感 · 时长可长 · 占屏可大。
Apple event / Awwwards SOTD · 2-3s
💬
Feedback · 反馈
响应用户操作。点赞 / 保存 / 错误 / 提交。**必须快** · 50-500ms · 别抢戏。
Linear 200ms · Apple 500ms
↔
Transition · 过渡
场景切换。页面跳转 / 模态弹出 / 标签页切换。维持连续感 · 不要"瞬切"或"慢吞吞"。
300-600ms fade+slide · 无中间态
🫁
Idle · 持续
待机状态。等待时的呼吸 / loading 的旋转 / 背景粒子漂浮。幅度极小但不停。
Breathing scale ±1% · Spinner
👆
Interact · 交互
悬停 / 按下 / 拖拽过程中。表达"可以操作"和"正在操作"。
Hover scale 1.02 · Press 0.95
🌙
Outro · 收尾
离场。关闭 / 删除 / 完结。对称于 Hero。好的收尾让作品有"呼出"感。
Fade+shrink · 或 drift-away

拆解 Apple-Like v4 · 看 5 维同时到位

一个"好动画"不是某一维做得好 — 是五维都选对了。这就是为什么 subagent 单看一两维容易翻车。

WHAT
爱心 SVG + "+1" 数字 + 小爱心粒子
LOOK
白底 + Apple Red radial gradient + 柔红 shadow + SF Pro 负字距
HOW
Spring k=260/d=14 + Anticipation + Stagger 粒子 + Arc 轨迹
WHY
反馈（确认点赞）+ 情感（被看到的激励）
WHEN
用户双击瞬间触发 · 完整弧 3s · loop

换任意一维 = 新作品。比如 LOOK 从 Apple 换成 Arc（紫粉蓝） = Arc 版 Like。HOW 从 spring 换成 linear = Linear 版 Like。全部 5 维同时换品牌 DNA = 风格切换（就是 compare-success/ 里 10 个）。

Module Architecture — 4 Layers

Think of the system like a building. Shared code is the foundation, core code is the structural frame, runtime modules are the rooms people move through, and app shells are the facade that exposes the building. The floor metaphor explains responsibility. The arrows show the real rule: dependencies move one way.

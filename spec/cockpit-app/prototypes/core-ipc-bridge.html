IPC Bridge — Rust ↔ JS Communication

Think of the bridge like a restaurant. JavaScript is the waiter taking the order, Rust is the kitchen doing the work, and the same waiter delivers the result back. The contract stays strict: one request envelope in, one response envelope out, across 31 core methods grouped by domain.

Design Notes

  • Panel 位置：编辑器右侧，替换现有 Exports Panel（360-420px 宽）
  • 数据源：perf.jsonl（已有）+ ProcessHandle 内存状态
  • Caller 标签：AI / UI / CLI 三种来源，颜色区分
  • 每条记录：文件名 + 时间 + 分辨率 + FPS + 时长 + 大小 + 速度
  • 展开详情：PID、起止时间、帧统计、编码器、CRF、音频、完整命令
  • 操作按钮：Open Video / Show in Finder / View Log / Copy Command / Retry
  • 顶部统计：累计时间、累计大小、平均速度
  • 队列状态：running(脉冲黄点) / queued(灰点+位置) / done(绿) / failed(红)
  • 按日期分组，sticky header

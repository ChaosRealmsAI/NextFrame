---

## 气质关键词

- **自嘲** — 源码被泄露了嘛、我数过了、这是我设计出来的"闲人勿扰"
- **技术自信** — 精确数字、源码出处、不打马虎眼
- **口语化** — 你用过豆包吧？你发微信的时候、打个比方
- **深入浅出** — 先类比再技术、每个抽象概念都先给日常对照

---

## 典型句式

### 开场钩子（3 种）
- "你以为 X？其实 Y。"
- "我数过了。{具体数字}。"
- "你用过 {常见产品}吧？那你想过没有..."

### 解释复杂概念
- "可以想象成 {日常事物}。你 {一件小事}，其实背后 {复杂机制}。"
- 铁律：**每个技术名词先给一个生活类比**，然后再给技术细节。

### 金句收尾
- "你写了 1 行，系统拼了 87 类。每一轮，重新拼一遍。"
- 结构：**具体数字 + 动作对比 + 周期强调**。

## 禁用词

- ❌ "相信"、"肯定"、"当然"（过度自信）
- ❌ "简单来说"（听起来居高临下）
- ❌ Emoji（面向技术观众）
- ❌ "令人震惊"、"颠覆性"（营销号腔调）
- ❌ "大家好，欢迎回到我的频道"（废话开场）

## 节奏

- **3Blue1Brown 式**：每 15-30 秒一个视觉锚点 + 一个类比锚点
- **衣戈猜想式**：高信息密度，每句话都加载新信息
- **忌留白**：没话说就切画面 / 换类比，不要口播空画面

## 类比库（这个主题常用）

| 技术概念 | 类比 |
|----------|------|
| API 槽位 | 快递包裹的几个格子 / 餐厅点菜的订单 |
| 静态缓存 | 手机预装 App / CDN |
| Memory | 旧笔记本 / 相册 |
| 压缩 | 真空压缩袋 / 行李收纳 |
| Agent Loop | 洗衣机 / 总经理实习生 |
| 工具调用 | 员工工具箱 / 快递分拣员 |
| Hook | 安检员 / 防狼门锁 |
| MCP | 外接硬盘 / 接口 |

**新概念必须找 1-2 个类比**。找不到 = 这个概念没想透。
